1.This programs perform insertion, deletion, search and display operation on treaps.
2.You need to select the operation which you want to perform on the treap.
Enter 'I' for insertion,'R' for deletion,'S' for search,'D' for display and E 
for exit.
3.If 'I'or'R' or 'S' is enetered then enter the key of the node to be inserted ,
deleted or searched.Then message will be displayed according to the input entered.
4.If 'D' is entered then the current treap structure is displayed,displaying key
and priority values of all the node and its children.
